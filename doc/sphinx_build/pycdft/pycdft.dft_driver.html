pycdft.dft_driver¶

PyCDFT is compatible with the following DFT codes:

  • Qbox (via client-server mode)

Set the constraint potential Vc in DFT code.

Given constraint potential Vc as an array of shape [vspin, n1, n2, n3], this method send the constraint potential to the DFT code.

Parameters
