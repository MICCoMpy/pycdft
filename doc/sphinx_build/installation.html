Installation¶

In order to install and use pyCDFT

  1. Compile DFT driver

pyCDFT is currently compatible with Qbox (used in client-server mode). Other DFT codes may be added in the future as needed.

  • Install Qbox

  • Install Qbox on Mira

PyCDFT has been tested with Python 3.6+, Qbox 1.67.4+. A version of Qbox that is capable of handling external potentials via the variable vext is required. Currently PyCDFT is compatible with cube and Function3d formats.

  1. Install PyCDFT

The basic command is:

$ python setup.py install

You may need to modify which directory to install in with “–prefix=/desired/installation/path/pycdft/”

E.g., in a bash environment

export PYTHONPATH="${PYTHONPATH}:/desired/installation/path/pycdft/:/desired/installation/path/pycdft/lib/python3.7/site-packages"
python setup.py install --prefix="/desired/installation/path"

Be sure to to have installed or have loaded Python v 3.6+, e.g.,

$ module load python
$ module list

Dependencies¶

The following dependencies are needed:

  • python

  • Jupyter

The following Python packages are needed, as included in setup.py install file:

  • numpy

  • pyFFTW

  • lxml

  • ase
